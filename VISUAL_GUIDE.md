## ✨ Special Effects

### Parallax Scroll (Hero)
```
Scroll Position 0:    Background at 0%
Scroll Position 50:   Background at 25%
Scroll Position 100:  Background at 50%
(Background moves slower than content)
```

### Intersection Observer
```
Element above viewport:  [Hidden]
Element enters viewport: [Fade in + Slide up]
Element in viewport:     [Visible]
```
